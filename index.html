案例開始：資料怎麼收？

Q1：哪裡一直重複？

每次都在問家屬一樣的基本資料。

✅ 用 Form 統一入口

🧠 觸發：表單一送出，GAS 就醒來

(💡 現場 Demo：填寫表單)

Q2：資料散在哪裡？

LINE、紙本找資料像考古。

✅ 自動收進 Sheet 集中

以前痛苦找訊息 ➔ 現在輕鬆查案件。

(💡 現場 Demo：欄位自動匯入)

收到之後，怎麼判斷？

Q3：固定的判斷？ (GAS 抓關鍵字亮燈)

「肯定」 ➔ 🟢 綠
「提告」 ➔ 🔴 紅
「其他」 ➔ 🟡 黃
如果 (有肯定內容) { 亮綠燈; }
否則如果 (有高風險) { 亮紅燈; }

(💡 放心不是要你手寫，AI 產出的指令大概長這樣)

Q4：要通知誰？ (依燈號寄信)

🟢 主管 (好評)
🟡 主管+社工+書記
🔴 主管+社工+管理層 (危機)
🧠 觸發：GAS 判斷完自動寄信

最後怎麼追蹤？

Q5：怎麼追蹤進度？

  • 狀態：⏳ 待處理 ➔ 🚧 處理中 ➔ ✅ 已結案
  • 自動封存：結案後自動收進倉庫表。
  • 手動變色：改燈號，自動換底色。
🧠 觸發：修改表格時 (onEdit)

(💡 現場 Demo：觸發自動搬移)

Q6：主管最後想看什麼？

主管不想看明細，只想看紅燈有幾件？

✅ 戰情室＋每天自動日報

🧠 觸發：每天下午五點 (Time-driven)

(💡 現場 Demo：展示 Dashboard)
